# Provenance

This skill was distilled from a Hermes prompt-writing bundle and a direct user correction: the failure was not ugly formatting, but prompts that sounded thorough while losing truth, intent, and model judgment.

The source review compared prompt-writing and skill-writing guidance from NeoLabHQ, Jeffallan, curiositech, VoltAgent, Composio, and Anthropic-style skill-creator material. Only the rules that changed behavior were kept:

- Treat freedom level as a first-class decision.
- Spend context like an attention budget.
- Avoid prompt stuffing and over-constraining.
- Use CLEAR as a pressure test, not a form.
- Improve prompts against realistic evals, not abstract theory.

The original Hermes runtime metadata, local paths, and research cache were intentionally left out of the global Codex skill.
