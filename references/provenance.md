# Provenance

This skill was distilled from a Hermes prompt-writing bundle and a direct user correction: the failure was not ugly formatting, but prompts that sounded thorough while losing truth, intent, and model judgment.

The source review compared prompt-writing and skill-writing guidance from NeoLabHQ, Jeffallan, curiositech, VoltAgent, Composio, and Anthropic-style skill-creator material. The skill was later revised for more capable models and real human-carried agent relays. Only rules that change behavior are kept:

- Calibrate autonomy without forcing a visible freedom label or rigid form.
- Preserve provenance across user-to-agent, agent-to-user, and agent-to-agent transfers.
- Separate paste-ready payloads from notes for the human courier.
- Match authorization and proof to the actual work.
- Spend context like an attention budget.
- Avoid prompt stuffing and over-constraining.
- Improve prompts against realistic evals, not abstract theory.

The original Hermes runtime metadata, local paths, and research cache were intentionally left out of the global Codex skill.
